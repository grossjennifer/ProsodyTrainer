# ProsodyTrainer
Open-source software supporting research, teaching, and intervention in English prosody and reading. 

The project includes:

- **Rhythm Reader** – a simple, accessible interface for exploring lexical stress and rhythm.
- **Rhythm Reader Pro** – an advanced interface for researchers, teachers, and clinicians that includes phrase structure, meter detection, editing tools, and diagnostic information.

The software is based on research in psycholinguistics, reading, and cognitive psychology and is intended for educational, clinical, and research applications.

Developed by Jennifer Gross, Grand Valley State University.
